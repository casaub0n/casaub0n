# 漫画としての感想

最初から（異様に詳細な）トップ私立男子校ホモソーシャル全開でぶっ飛ばしていて面白かった。
最近話題になったメンズコーチだが、コーチングの内容そのものが間違っているのがこの漫画で分かってくる。
**1、2巻で出でくる中で大切なのは以下の要素**

- 過去は忘れろ
- 出口をまず考えろ
- ルールの奴隷になれ（無駄なことを考えるな）
- 体力は大事

今は高速取引とAIの時代なので数字（結果）に拘るならコンピュータサイエンス太郎になりなさいで終わると思う。昔の人にとってはつまらない時代だ。

# 感想

バンコクは物価、円安で高そう。とにかく車やバスが込みまくっているので電車とかがいいのかもしれない。最近BTSもけっこう拡張しているとyoutubeで見た。
中華街のスープ、食器購入、カフェ巡りやアートとかが良さそう。タイ北部のアラビカ種でハンドドリップはやってみたい。タイ料理の歴史も含めた講義やハンズオンをやりたい。Yシャツのオーダーメイドとか面白そう。
アユタヤとか北部の方が落ち着いているし、歴史もあるので、そっちに移動してのんびりしたい。
